# Bank Management System (ATM Simulator)

## Description
The Bank Management System is a Java-based ATM Simulator application developed using
Java Swing and MySQL. It simulates real-world ATM functionalities and allows users
to perform secure banking operations through a graphical user interface.

## Technologies Used
- Java
- Java Swing (AWT)
- JDBC
- MySQL

## Features
- New User Registration (Multi-page Signup Form)
- Card Number and PIN Generation
- Secure Login
- Cash Deposit
- Cash Withdrawal
- Balance Enquiry
- Mini Statement
- Fast Cash
- PIN Change

## Database Design
The project uses MySQL as the backend database.
Tables included:
- signup
- signuptwo
- signupthree
- login
- bank

SQL commands are available inside the `database` folder.

## How to Run the Project
1. Install JDK 8 or above
2. Install MySQL Server
3. Create database `bankmanagementsystem`
4. Execute the SQL file from the `database` folder
5. Update database username and password in `Conn.java`
6. Run `Login.java`
